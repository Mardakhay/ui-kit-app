# UI Kit App

Practice project for building and testing reusable UI components with React and TypeScript.

## About

This project is used to practice UI development and component architecture.  
It includes both custom-built components and Material UI examples to compare different approaches.

## Tech Stack

- React
- TypeScript
- Vite
- Material UI

## What’s Included

- Custom components:
  - Button (variants, sizes, loading, icons)
  - Input (validation, variants, helper text)
  - Card (compound component pattern)
- Showcase pages:
  - Custom UI Kit
  - Material UI
- Basic layout and styling system
